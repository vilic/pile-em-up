<h1 align="center">Routra</h1>
<p align="center">Just another state router with transition support.</p>

## Installation

```bash
npm install routra
```

## Usage

```ts
import {routra} from 'routra';

const router = routra({
  user: {
    profile: true,
    settings: true,
  },
});
```
